In this scenario, we will demonstrate how to leverage the SPFx project upgrade Language Model Tool within GitHub Copilot Chat to upgrade a SharePoint Framework (SPFx) project to the latest version. This process simplifies the upgrade by utilizing AI capabilities to handle the necessary changes for you allowing you to focus on other important tasks.
SPFx Project Upgrade will first generate the upgrade report, similar to using [upgrade action](../features/actions#upgrade-project), and uses GitHub Copilot to apply the changes automatically.
